# Millan Sawhney Portfolio Website

This is a clean, modern, fully responsive portfolio website for a pre-med student, designed for easy deployment to GitHub Pages.

## Features

- Sections: About, Experience, Research Goals, Contact
- Hero section with introduction and call-to-action buttons
- Experience shown in modern, clean cards
- Research interests and current goals clearly highlighted
- Mobile-friendly navigation with hamburger menu and smooth scrolling
- Professional and polished style using only HTML, CSS, and minimal JavaScript
- Easy to customize and extend with your own projects/publications

## Deployment Instructions (GitHub Pages)

1. **Create a GitHub repository** (public or private).

2. **Add the website files:**

   - Upload `index.html`
   - Upload `style.css`
   - Upload `script.js`
   - Upload `README.md`

3. **Enable GitHub Pages:**

   - Go to your repository on GitHub.
   - Click on "Settings".
   - In the left sidebar, choose "Pages".
   - Under "Source", select the branch (usually `main` or `master`) and root directory (`/`).
   - Click "Save".
   - After a few moments, GitHub will provide a URL where your website is live.

4. **Access your portfolio at the provided URL.**

## Customization

- Replace the text content in `index.html` with your updated information.
- Add research projects, publications, or links in the placeholder area under Research Interests.
- Customize colors or fonts by modifying `style.css`.

## Notes

- Tested on modern browsers.
- Designed for clarity and professional presentation.
- Minimal dependencies; no build tools required.
- Works well on desktop and mobile devices.
